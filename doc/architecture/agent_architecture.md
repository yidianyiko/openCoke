## 2. 消息处理主流程

```
用户发送消息: "一分钟后提醒我上床"
                │
                ▼
┌───────────────────────────────────────────────────────────────┐
│  Phase 1: PrepareWorkflow                                      │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ Step 1: QueryRewriteAgent                                │  │
│  │ - 输入: 用户消息 + 历史对话                               │  │
│  │ - 输出: 检索关键词 (CharacterSettingQuery, UserQuery等)   │  │
│  │ - LLM调用: 1次 ✅                                        │  │
│  └─────────────────────────────────────────────────────────┘  │
│                          │                                     │
│                          ▼                                     │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ Step 2: ReminderDetectAgent                              │  │
│  │ - 输入: 用户消息                                         │  │
│  │ - 工具: reminder_tool                                    │  │
│  │ - 输出: 创建提醒 (如果检测到提醒意图)                     │  │
│  │ - LLM调用: 1-2次 ⚠️ (每条消息都执行，即使无提醒意图)     │  │
│  └─────────────────────────────────────────────────────────┘  │
│                          │                                     │
│                          ▼                                     │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ Step 3: ContextRetrieveAgent ⚠️ 问题点                   │  │
│  │ - 输入: QueryRewrite 的输出                              │  │
│  │ - 工具: context_retrieve_tool                            │  │
│  │ - 输出: 检索到的上下文 (角色设定、用户资料等)             │  │
│  │ - LLM调用: 2次 ❌ (完全不必要，参数已确定)               │  │
│  └─────────────────────────────────────────────────────────┘  │
└───────────────────────────────────────────────────────────────┘
                │
                │ ← 检测点1: 检测新消息
                ▼
┌───────────────────────────────────────────────────────────────┐
│  Phase 2: StreamingChatWorkflow                                │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ ChatResponseAgent (流式)                                 │  │
│  │ - 输入: 用户消息 + 检索上下文 + 历史对话                  │  │
│  │ - 输出: 多模态回复 (文本/语音/图片)                       │  │
│  │ - LLM调用: 1次 ✅                                        │  │
│  │ - 流式输出，每条消息立即发送                              │  │
│  └─────────────────────────────────────────────────────────┘  │
└───────────────────────────────────────────────────────────────┘
                │
                │ ← 检测点2: 每条消息发送后检测新消息
                ▼
┌───────────────────────────────────────────────────────────────┐
│  Phase 3: PostAnalyzeWorkflow (可被跳过)                       │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ PostAnalyzeAgent                                         │  │
│  │ - 输入: 本轮对话内容                                      │  │
│  │ - 输出: 记忆更新 (用户资料、关系描述等)                   │  │
│  │ - LLM调用: 1次 ✅                                        │  │
│  └─────────────────────────────────────────────────────────┘  │
└───────────────────────────────────────────────────────────────┘
```

## 3. Agent 清单

| Agent 名称 | 所属 Workflow | 功能 | LLM 调用次数 | 使用的 Tool |
|-----------|--------------|------|-------------|-------------|
| QueryRewriteAgent | PrepareWorkflow | 语义理解，生成检索关键词 | 1 | 无 |
| ReminderDetectAgent | PrepareWorkflow | 检测提醒意图，创建提醒 | 1-2 | reminder_tool |
| ContextRetrieveAgent ⚠️ | PrepareWorkflow | 检索上下文 | 2 | context_retrieve_tool |
| ChatResponseAgent | StreamingChatWorkflow | 生成多模态回复 | 1 | 无 |
| PostAnalyzeAgent | PostAnalyzeWorkflow | 总结对话，更新记忆 | 1 | 无 |

**总计：一条消息需要 6-7 次 LLM API 调用**

## 4. Tool 清单

| Tool 名称 | 功能 | 被哪个 Agent 使用 |
|----------|------|------------------|
| reminder_tool | 提醒管理 (创建/更新/删除/列表) | ReminderDetectAgent |
| context_retrieve_tool | 向量检索 (角色设定、用户资料、知识库) | ContextRetrieveAgent |
| voice_tools | 语音合成 (TTS) | ChatResponseAgent (间接) |
| image_tools | 图片处理 | ChatResponseAgent (间接) |
| album_tools | 相册管理 | - |

## 5. 问题分析 & 优化建议

### 5.1 当前问题

```
当前流程 (一条消息):
┌─────────────────────────────────────────────────────────────┐
│ QueryRewrite → ReminderDetect → ContextRetrieve → Chat → Post│
│     1次           1-2次            2次           1次    1次  │
│                                    ↑                         │
│                              完全不必要！                     │
└─────────────────────────────────────────────────────────────┘
总计: 6-7 次 LLM 调用 + 5 次 Telemetry 上报
预计耗时: 50-60 秒
```

### 5.2 优化后流程

```
优化后流程 (一条消息):
┌─────────────────────────────────────────────────────────────┐
│ QueryRewrite → [ReminderDetect] → ContextRetrieve → Chat → Post│
│     1次         0-1次 (按需)      0次 (直接调用)   1次    1次  │
│                    ↑                    ↑                     │
│              只在有提醒关键词时      直接调用 Tool 函数         │
└─────────────────────────────────────────────────────────────┘
总计: 3-4 次 LLM 调用 + 关闭 Telemetry
预计耗时: 20-25 秒
```

### 5.3 具体优化项

| 优化项 | 当前 | 优化后 | 节省 |
|-------|------|-------|------|
| ContextRetrieveAgent | 2次 LLM 调用 | 0次 (直接调用函数) | ~5秒 |
| ReminderDetectAgent | 每条消息都执行 | 仅关键词匹配时执行 | ~3秒 (大部分消息) |
| Telemetry | 5次上报 | 关闭 | ~2秒 |
| **总计** | **50-60秒** | **20-25秒** | **~30秒** |

## 7. 文件结构

```
agent/
├── runner/
│   ├── agent_runner.py          # 主入口，启动消息轮询
│   ├── agent_handler.py         # 消息处理主逻辑，调度 Workflow
│   ├── agent_background_handler.py  # 后台任务 (提醒触发等)
│   └── context.py               # 构建 session_state
│
├── agno_agent/
│   ├── agents/
│   │   └── __init__.py          # 定义所有 Agent 实例
│   │
│   ├── workflows/
│   │   ├── prepare_workflow.py      # Phase 1: 准备阶段
│   │   ├── chat_workflow_streaming.py  # Phase 2: 流式回复
│   │   └── post_analyze_workflow.py # Phase 3: 后处理
│   │
│   ├── tools/
│   │   ├── reminder_tools.py        # 提醒管理
│   │   ├── context_retrieve_tool.py # 向量检索
│   │   ├── voice_tools.py           # 语音合成
│   │   └── image_tools.py           # 图片处理
│   │
│   └── schemas/
│       ├── query_rewrite_schema.py  # QueryRewrite 输出格式
│       ├── chat_response_schema.py  # ChatResponse 输出格式
│       └── post_analyze_schema.py   # PostAnalyze 输出格式
│
└── prompt/
    ├── system_prompt.py         # System Prompt
    ├── chat_taskprompt.py       # Task Prompt
    └── chat_contextprompt.py    # Context Prompt 模板
```
