# ADR Index

Architectural Decision Records capture durable choices that affect repository
structure, workflow, or long-term operating conventions.

## When To Write An ADR

Write an ADR when a decision changes:

- repository structure
- workflow lifecycle
- verification model
- task or plan conventions
- automation boundaries
- coordination model

Do not write an ADR for routine task-specific implementation choices.
