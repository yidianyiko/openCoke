# Task: Kap Koala Mascot

- Status: In Progress
- Owner: Codex
- Date: 2026-04-24

## Goal

Replace Kap's current letter-style homepage mascot and outward-facing brand icon treatment with a hand-drawn sticker-style koala.

## Scope

- In scope:
  - homepage hero mascot
  - outward-facing brand icon surfaces in gateway web
  - shared public shell brand mark
  - customer-facing shell brand mark
  - global page brand mark
  - `logo.png` and `favicon.ico`
- Out of scope:
  - backend or route changes
  - admin behavior changes
  - broader mascot pose library work beyond the hero + badge pair

## Acceptance Criteria

- homepage hero no longer uses the `K` / `AI` block mascot treatment
- homepage hero renders a hand-drawn koala mascot
- shared outward-facing brand entrypoints render a koala badge alongside the Kap wordmark
- public metadata and browser icon use the koala badge asset
- targeted brand tests, full gateway web tests, and production build all pass
